# GT updated
## 📝description
ファイル操作を自動化するためのシンプルなスタンドアロンデスクトップアプリケーション。利用にはディレクトリパスと操作するファイル一式が入ったzipファイルが必要です。ファイル操作はzipファイルに配置されたmanifest.jsonに基づきファイルの追加や削除・上書きされます。私はこのアプリをMinecraftのマルチサーバーでクライアントサイドのModpackの自動更新ツールとして開発/使用しました。  
主な機能：  
- ワンクリック・アップデート
- ローカル実行、ダウンロード実行が可能
- クロスプラットフォーム対応(Windows10/11, ubuntu, macOS)

## 🤔How to Use?
複雑な設定は必要ありません。  
1.ゲームディレクトリを選択してください（次からは必要ありません）  
2.アップデートを確認ボタンをクリックしてください
